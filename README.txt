SuperMail is a term project for CPSC 304 (Relational Databases)

This project uses PHP and MySQL.

To setup:
1. Download XAMPP at https://www.apachefriends.org/download.html and setup the MySQL database.

2. Locate the filepath of XAMPP, under the application folder there should be a folder called htdocs. You want 
to clone this project under htdocs.

3. After setting up the MySQL database, go to http://localhost/phpmyadmin/ , create a new database called 
'SuperMail'

4. In the cloned project, go to SuperMail/sql/create.sql , copy all text from the file and paste it in the 
MySQL database for SuperMail, then hit 'Go'. The database should now be loaded in MySQL, check on the left bar.

5. Go to http://localhost/SuperMail/index.html , the website should be running and ready to be used.
